Tee Times

  • Tools: Python, Firefox Developer, Launch Agents
  • Production Date: January 2026
  • I built a Python automation script that monitors a city golf website for newly released tee times. Because the site uses dynamic, client‑side rendering, it requires more than simple HTML scraping, so I incorporated libraries that can handle asynchronous content and API‑driven updates. The script gathers all available tee times and compares them against a stored JSON file that tracks what’s already been seen. When new openings appear, it automatically sends me an email alert so I can act quickly. These visuals walk through the scraping workflow, the JSON‑based change detection, and the notification system.

Nasa Space Data

  • Tools: Python, Selenium, Jupyter Notebooks
  • Production Date: March 2021
  • I created a Python web‑scraping tool that pulls structured planetary data from NASA’s website using an HTML parser. Since the site exposes its information through traditional markup rather than dynamic rendering, I focused on clean parsing logic and reliable extraction patterns. The workflow uses a mix of Python libraries to fetch pages, parse key attributes, and organize the results into a usable format. All development and testing took place in Jupyter notebooks, which made it easy to iterate quickly and validate each stage of the scraper.
